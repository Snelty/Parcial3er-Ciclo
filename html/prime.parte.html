PRIMERA PARTE: CINCO PREGUNTAS

1. ¿Qué es Bootstrap?

Bootstrap es un framework de desarrollo web front-end de código abierto. Proporciona plantillas HTML, CSS y JavaScript diseñadas para facilitar la creación de sitios

2. ¿Para qué utilizamos Bootstrap?

Utilizamos Bootstrap principalmente para:

  • Acelerar el desarrollo: Evita tener que escribir mucho código CSS y JavaScript desde cero.
  • Diseño responsivo: Crear sitios web que se adaptan automáticamente a diferentes tamaños de pantalla (móviles, tablets, desktops).
  • Consistencia visual
  • Compatibilidad entre navegadores: Ayuda a que el sitio se vea y funcione bien
  • Facilitar la colaboración
  • Prototipado rápido

3. ¿Para qué utilizamos los grids (rejillas)?

En el contexto del diseño web (y especialmente en Bootstrap), utilizamos los sistemas de grids o rejillas para:

  • Organizar el contenido: Dividir la página en filas y columnas para estructurar y alinear los elementos de manera ordenada.
  • Crear diseños responsivos: Permiten que el layout se ajuste fluidamente a diferentes anchos de pantalla. Por ejemplo, un diseño de 3 columnas en escritorio puede pasar a 1 columna en móvil.
  • Facilitar el diseño complejo

4. ¿Cuál es la estructura de un botón y qué se puede realizar con los mismos?

La estructura básica de un botón en HTML suele ser el elemento <button> o un enlace <a> estilizado como botón. Ejemplo:

<button type="button" class="btn btn-primary">Texto del Botón</button>

En Bootstrap, se añaden clases como .btn y clases de estilo (.btn-primary, .btn-secondary, etc.).

¿Qué se puede realizar con los botones?

  • Enviar formularios: (type="submit") Para mandar datos a un servidor.
  • Ejecutar acciones JavaScript
  • Navegación: Si se usa un elemento <a>, pueden llevar a otras páginas o secciones.
  • Restablecer formularios: (type="reset") Para limpiar los campos de un formulario.
  • Controlar elementos de la interfaz: Abrir modales, mostrar/ocultar contenido, controlar carruseles, etc.
  • muchomas: Se puede hacer mucho con botones.

5. ¿Los carruseles cuántos casos tienen?

  • Contenedor principal: El elemento que envuelve todo el carrusel (ej. .carousel).
  • Indicadores: Pequeños puntos o líneas que muestran cuántos "slides" hay y cuál está activo (ej. .carousel-indicators).
